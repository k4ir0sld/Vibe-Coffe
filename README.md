# Vibe-Coffe

## Sobre o Projeto

A Vibe Coffee é uma cafeteria temática inspirada na estética e nostalgia dos anos 80. Este projeto foi desenvolvido como parte do Projeto Integrado do curso de Desenvolvimento de Sistemas, envolvendo as disciplinas de Projetos, Frontend, Backend e Banco de Dados.

O sistema consiste em uma plataforma web para divulgação e gerenciamento do catálogo da cafeteria, permitindo que clientes visualizem os produtos disponíveis e que o administrador realize a gestão completa do cardápio por meio de um painel administrativo.

---

## Objetivo do Repositório

Este repositório tem como objetivo centralizar todas as informações, documentações e links relacionados ao projeto Vibe Coffee.

Ele funciona como ponto principal de acesso para professores, avaliadores e integrantes da equipe, reunindo os artefatos produzidos durante o desenvolvimento do sistema.

---

## Funcionalidades do Sistema

### Área Pública

* Visualização do catálogo de produtos;
* Pesquisa de produtos por nome;
* Filtragem por categorias;
* Visualização detalhada dos produtos;
* Interface responsiva para dispositivos móveis e desktops.

### Área Administrativa

* Autenticação de administrador;
* Cadastro de produtos;
* Edição de produtos;
* Exclusão de produtos;
* Controle de disponibilidade;
* Gerenciamento de categorias.
